Clarity: um modo calmo de ver seus comandos.

Clarity não substitui npm, git ou docker. Ele só envolve o comando, captura o log, remove o ruído e devolve um bloco resumido com causa + próximo passo.

O mesmo comando de sempre, só que legível. Se precisar do caos completo, rode com --full e pronto.

O que o Clarity faz?

> Menos ruído

Agrupa warnings repetidos e deixa um badge indicando quantos foram silenciados.

✓ Resumo humano

Transforma logs em poucas linhas com contexto, status e próximo passo explícito.

! Sem magia

Executa o comando real, preserva exit code e não mexe no PATH. É só um wrapper.

≡ Log completo

clarity run --full despeja a saída original sempre que você quiser.

Como funciona?

Troque o prefixo

Faça clarity npm install em vez de npm install. Nada mais.

Clarity captura e organiza

O stdout/stderr é salvo, deduplicado e condensado em um bloco calmo.

Você escolhe o nível

Fique no resumo, abra quick previews ou despeje tudo com --full.

Instalação

# instale globalmente
npm install -g clarityterm

# use como prefixo
clarity npm install
clarity git pull
clarity docker build

Clarity só envolve o comando e formata o output. Nada de shims mágicos, nada de build step secreto.
